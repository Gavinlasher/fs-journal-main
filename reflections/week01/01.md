## Questions
1. In your own words, why do we use Git?
We use git to track and save our code. So that it can be put into github to be worked on with others or been seen by others.
2. What are the benefits of having multiple branches?
benefit of have multiple branches would be to test different code also to have different back ups of the code. Developers can also work on projects together.
3. What is the difference between Git and GitHub
git is tracking system and to save code to later be uploaded to github. Which is a cloud service for other devs to see and retrieve.
